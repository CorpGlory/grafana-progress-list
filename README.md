## About

Show list of progress items by mapping your data.

## Installation

```
cd $GRAFANA_HOME/data/plugins
git clone https://github.com/CorpGlory/grafana-progress-list.git
```

then restart your Grafana server

## Build

```
npm install
npm run build
```
